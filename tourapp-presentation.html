Chapter 01

Luồng Khởi Động App

Backend Startup (MauiProgram.cs)

Tầng 1 — WebView Android Config

Cấu hình trước khi build: JavaScriptEnabled = true, DomStorageEnabled = true, MixedContentMode = AlwaysAllow, LayerType.Hardware. Bắt buộc để Goong Maps JS chạy được.

Tầng 2 — DI Container

Đăng ký Singleton: ApiClientService, LocationService, GeofenceService. Đăng ký Transient: MapPage.

Tầng 3 — AppShell → MapPage

Shell route mặc định đến MapPage. Constructor inject 3 services, đăng ký 2 events.

Tầng 4 — OnAppearing() [Critical Path]

InitializeAsync() → GetAllPOIsAsync() → LoadMap() inject HTML+Goong JS vào WebView → StartTracking() bắt đầu poll GPS.

Frontend Startup Flow

App Launch→ MauiProgram DI→ AppShell→ MapPage→ Load POIs (mock)→ Goong Map Render→ GPS Tracking
✅ App Khởi Động Tối Ưu: Dữ liệu POI được tải bất đồng bộ (Async) từ Backend REST API. Giao diện thiết kế đảm bảo GPS chạy Background Thread không blocks UI, bypass timeout.

Chapter 02

GPS + Geofence + Audio Narration

LocationService poll GPS mỗi 5 giây. Mỗi update kích hoạt CheckGeofences() — tính Haversine đến từng POI, debounce 2 phút cooldown, trigger TTS + UI khi vào zone.

Thuyết Minh Tự Động & Chống Spam

Hệ thống hỗ trợ Debounce & Cooldown: Giới hạn 2 phút/POI để tránh lặp thoại nếu user đứng yên. Geofence trigger tự động khi Location rơi vào Radius của POI.

🔥 Tính Năng Mới - Quét QR Bypass: Dán QR tại điểm trạm. User quét QR -> Request Backend lập tức -> Trả về URL MP3/Script Audio -> Xóa hàng đợi và phát đè luôn (Bypass GPS Radius).

Luồng Geofence Engine (4 Bước)

Giai đoạn 1 — GPS Collection (LocationService)

GetLocationAsync() mỗi 5s trên background thread → MainThread.BeginInvokeOnMainThread() phát LocationChanged event an toàn.

Giai đoạn 2 — Zone Detection (GeofenceService)

CheckGeofences(pos): Location.CalculateDistance() tính khoảng cách. ≤ radius → kiểm tra cooldown → TriggerNarration(poi). Chỉ trigger 1 POI/lần (break).

Giai đoạn 3 — Audio Decision

Cooldown 2 phút/POI: nếu poi.PoiId == _lastSpokenPoiId và chưa qua 2 phút → bỏ qua. Tránh thuyết minh lặp lại khi đứng yên.

Giai đoạn 4 — Audio Playback + UI

PoiTriggered event → ShowPoiDetails(poi) mở bottom sheet. Song song: SpeakNarrationAsync() → TextToSpeech.Default.SpeakAsync().

📏 Radius per POI

  • Ốc Oanh: 30m (Priority 1)
  • Lẩu Bò Khu Vực: 25m (Priority 1)
  • Trà Sữa Vĩnh Khánh: 20m (Priority 2)
  • Default model: 80m

⏱️ Timing Constants

  • GPS Interval: 5s
  • Geolocation timeout: 5s
  • Audio cooldown/POI: 2 phút
  • Max trigger/cycle: 1 POI

🔗 JS Bridge (Map ↔ C#)

C# → JS: MapWebView.Eval("updateUserLocation(...)") cập nhật dot vị trí, map.flyTo()

JS → C#: Click marker → poi://selected?id=1 → OnMapWebViewNavigating()

Chapter 03

POI Content — Vòng Đời Dữ Liệu

📖 Load POI (Startup)

GetAllPOIsAsync() → mock 3 POI hardcode (sau này: GET /api/pois từ MySQL). Dữ liệu load 1 lần khi app mở, lưu trong _pois list.

🗺️ Render Markers

Serialize List<POI> → JSON → inject vào HTML Goong Map. JS loop tạo goongjs.Marker cho từng POI với emoji 🍽️.

🔍 Search POI

1. Tìm trong _pois local (PoiName contains). 2. Nếu không có: Goong AutoComplete API → Place Detail API → map.flyTo().

⭐ Favorites

Lưu qua Preferences.Default với key fav_{poiId}. Không cần DB — persist ngay trên device giữa các session.

3-Tier Content Fallback (kế hoạch)

Tier 1
MySQL REST API (online) GET /api/pois → dữ liệu đầy đủ, ảnh URL, audio_url từ server.
Tier 2
SQLite Local Cache (offline) Lưu POI sau khi fetch lần đầu → mở lại app không cần mạng vẫn có dữ liệu.
Tier 3
Mock Data Hardcode (fallback cuối) USE_MOCK_DATA = true — 3 POI demo Vĩnh Khánh. Luôn hoạt động kể cả khi backend chưa xong.

Chapter 07

Offline Capability

✅ Đã offline: TTS

TextToSpeech.Default dùng Android TTS engine tích hợp. Không cần mạng để phát thuyết minh. Phiên bản vi-VN tải sẵn trong device.

✅ Đã offline: POI Data

Mock data hardcode → luôn có dữ liệu kể cả mất mạng. Khi có MySQL: cần thêm SQLite cache lần đầu load.

✅ Đã offline: Favorites

Preferences.Default lưu local trong app storage. Persist giữa các lần mở app, không cần mạng.

❌ Chưa offline: Bản đồ

Goong Maps JS load tile từ CDN. Mất mạng = bản đồ trắng. Giải pháp: cache tile lần đầu hoặc dùng map tĩnh fallback.

4 Lớp Defense (hiện tại)

Lớp Thành phần Trạng thái
1 — Audio MAUI TextToSpeech (device TTS) ✅ Offline hoàn toàn
2 — POI Data Mock data / SQLite local cache ✅ Mock xong, SQLite TODO
3 — Favorites MAUI Preferences local storage ✅ Hoàn chỉnh
4 — Bản đồ Goong Maps JS (CDN) ❌ Cần internet
